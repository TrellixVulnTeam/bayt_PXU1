Upgrade vBBU or vNode Devices

Once a software image is uploaded to the PW EMS, you can upgrade one or more vBBU or vNode devices with it.
  1. In the left navigation pane, click Devices > Upgrade.
  2. In the Device Upgrade and Rollback window Image Inventory section, find the row of the upgrade software image that you want to use to upgrade the device and click the dotted icon for this row in the Actions column.
  3. Select Download from the drop-down list as shown below.
  4. In the Download Upgrade Image dialog box, check the checkbox next to the desired device(s) as shown below:
  5. Click Download. The device image that you previously uploaded to the PW EMS is now downloaded to the vBBU or vNode device and is ready to be activated.
  6. In the Image Inventory section, select the row of the upgrade software image that you downloaded to the device(s) and click the dotted icon again from the Actions column.
  7. Click Activate.
  8. In the Activate Upgrade Image dialog box, check the checkbox next to the devices for which you previously downloaded the new software and click Activate to reboot and activate these devices, as shown below.
