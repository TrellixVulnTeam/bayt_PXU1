Perform a Bulk Reboot Action on Devices

Use this task to perform a bulk reboot action on one or more vBBU, vNode, and cells.

  1. Use any of the following ways to access the Device Summary window:
    • Log into the PW EMS, this window displays by default.
    • From the navigation bar, click Dashboard.
    • From the navigation bar, click Devices > Device Summary.
  2. In the Device Summary window, check the check box next to any vBBU or vNode device(s) on which you want to perform a bulk reboot action.
    NOTE: You can select all devices in the Device Summary table for this option by clicking the Actions checkbox.
  3. Click BULK Actions. In the drop-down list that appears, select from the following reboot actions.

  7. In the Confirm Reboot dialog box, click CONFIRM.
