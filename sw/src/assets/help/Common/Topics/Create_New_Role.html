Create a New User Role for Your Organization

An assigned user role is a collection of permitted operations that can be performed by this user. For example, a certain user is permitted to perform the operation of managing alarms. There is no limit for how many roles can be created for an organization.

  1. Click the Navigation Bar menu and select Administration > RBAC.
  2. Select the appropriate organization from the drop down list at the top of the Managing Organization window.
  3. Select the ROLES tab and click CREATE NEW ROLE FOR:<Organization>.
  4. In the Create New Role for:<Organization> dialog box complete the following fields:

    Field

    Description
    Role Name The name of the user role. For example, Network System Engineer.
    Role Description A detailed description of the user role. For example, "This user installs and configures intelligent dashboards and O-RAN elements on the system, which includes vBBU access nodes, the HNG, EPC, and IMS systems. This user also operates, manages, and scales the network to add additional capacity, coverage, and enable different functions."
    Default Access Level drop-down list

    Select from the following access-levels:

    • Read/Write - User has permissions for everything associated with this role.
    • Write - User can perform assigned actions for this role.
    • Read - User can read only.
    • No Access - User belonging to this role cannot access and see parameters associated with certain actions.
  5. Click CREATE THE ROLE.
