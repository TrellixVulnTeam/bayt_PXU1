Organizations

An organization consists of users and vendor devices that can be a separate, individual company or entity or be one of many teams of users that belong to a single company or entity. Multiple organizations can be configured on the PW EMS system.

The PW EMS is configured with a default organization that can later be renamed. This default organization has a default "super" administrator user role and password. This user is enabled to manage all resources within the system, and is the only user during the initial system deployment.
