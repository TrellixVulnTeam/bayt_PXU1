Send vNode Information from the vBBU to PW EMS

Use the vBBU Web Commisioning Tool on the vBBU server Web UI to send a vNode inventory report from the vBBU that informs the PW EMS about its vNodes so that these vNodes can be configured on the PW EMS.

The vNode inventory report can be sent from the vBBU when the following actions are performed on the vBBU first through the Web Commissiong Tool:

  • The vBBU itself must be configured, which includes pre-provisioning the HNG, PW EMS, IPSec tunnel configuration and so on.
  • Access devices such as the RRH must be discoverd by the vBBU.
  • vNodes must be pre-provisioned on the vBBU and discovered access devices such as the RRH must be associated with the vNodes. For example, a vNode can be pre-provisioned for an RRH that supports 2G and another vNode can be created for the same RRH that also supports 4G.
  • Once the vNodes are pre-provisioned on the vBBU, they are ready to be reported to the PW EMS.

Refer to the Virtual Baseband Unit Server Commissioning Guide for more information.
