Manage System Settings for Devices

Use the following tasks to manage system settings for devices that connect with the PW EMS, which include system log levels, cell KPI 2G and 4G levels for PW EMS Network Monitoring, manage cell performance algorithm configurations, configure a syslog server, and enter Northbound SNMP information.

  • Configure the KPI Collection Interval for Cells
    Use this task to configure how often Key Performance Indicator (KPI) parameters are collected for 2G and 4G cells.
  • Configure System Logging Settings
    Use this task to configure system logging settings for troubleshooting and maintaining the PW EMS.
  • Manage Cell Algorithms
    Use this task to enable or disable the Automatic Neighbor Relations (ANR) algorithms that are used for the main LTE cell configuration for all devices connected to the PW EMS. The Self-Organizing Network (SON) uses the Automatic Neighbor Relation (ANR) feature to generate relations between radio network entities (neighboring cells) automatically.
  • Configure the System Log Server
    Use this task to configure Northbound sytem log (syslog) server connectivity to the PW EMS.
  • Configure the System SNMP User Profile
    Use this task to configure the system SNMP user profile used to authenticate SNMP users on the PW EMS.
