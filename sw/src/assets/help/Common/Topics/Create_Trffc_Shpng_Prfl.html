Create a Traffic Shaping Profile

Use this task to create a traffic shaping profile necessary for categorizing uplink access network traffic.

  1. In the navigation bar, click Profiles > Summary.
  2. In the Profiles window, click the System Profiles tab.
  3. Click +Create.
  4. In the Create System Profile window, enter the name of this profile in the Profile Name field.
  5. Select TrShapingUplink from the Profile Type drop-down list, and complete the following fields:
    NOTE: Use the Name field to search for a specific parameter on this page.

    Field

    Description
    Name Enter a name for all the uplink traffic.
    Bandwidth The default uplink bandwidth is 5 Mbps. Enter the uplink bandwidth value from 1 to 2000 Mbps.
    Total Bandwidth The default total uplink bandwidth is 120. Enter the total bandwidth value threshold from 1 to 2000 Mbps.
  6. Click Control to configure bandwidth settings for Differentiated Services Code Point (DSCP) profiles for backhaul network data traffic identified for congestion-handling purposes by completing the following fields:

    Field

    Description
    Bandwidth Enter the allowed bandwidth threshold from 0 to 90 percent that generates an alarm for the control traffic type if crossed.
    Raise Alarm Enter the raised alarm threshold limit from 0 to 100 percent for the control traffic type if crossed.
    Clear Alarm Enter the clear alarm threshold limit from 0 to 100 percent, where a user-defined script or action is initiated for the control traffic type if crossed.
  7. Click Signaling to configure bandwidth settings for DSCP profiles for backhaul network data traffic identified for signaling purposes by completing the following fields:

    Field

    Description
    Bandwidth Enter the allowed bandwidth threshold from 0 to 90 percent that generates an alarm for the signaling traffic type if crossed.
    Raise Alarm Enter the raised alarm threshold limit from 0 to 100 percent for the signalling traffic type if crossed.
    Clear Alarm Enter the clear alarm threshold limit from 0 to 100 percent, where a user-defined script or action is initiated for the signalling traffic type if crossed.
  8. Click UMTSVoice to configure bandwidth settings for DSCP profiles for backhaul network UMTS voice traffic.

    Field

    Description
    Bandwidth Enter the allowed bandwidth threshold from 0 to 90 percent that generates an alarm for the UMTS voice traffic type if crossed.
    Raise Alarm Enter the raised alarm threshold limit from 0 to 100 percent for the UMTS voice traffic type if crossed.
    Clear Alarm Enter the clear alarm threshold limit from 0 to 100 percent, where a user-defined script or action is initiated for the UMTS voice traffic type if crossed.
  9. Click UmtsData to configure bandwidth settings for DSCP profiles for backhaul network UMTS data traffic.

    Field

    Description
    Bandwidth Enter the allowed bandwidth threshold from 0 to 90 percent that generates an alarm for the UMTS data traffic type if crossed.
    Raise Alarm Enter the raised alarm threshold limit from 0 to 100 percent for the UMTS data traffic type if crossed.
    Clear Alarm Enter the clear alarm threshold limit from 0 to 100 percent, where a user-defined script or action is initiated for the UMTS data traffic type if crossed.
  10. Click LteData to configure bandwidth settings for DSCP profiles for backhaul network LTE data traffic.

    Field

    Description
    Bandwidth Enter the allowed bandwidth threshold from 0 to 90 percent that generates an alarm for the LTE data traffic type if crossed.
    Raise Alarm Enter the raised alarm threshold limit from 0 to 100 percent for the LTE data traffic type if crossed.
    Clear Alarm Enter the clear alarm threshold limit from 0 to 100 percent, where a user-defined script or action is initiated for the LTE data traffic type if crossed.
  11. Click Create+.
