View vBBU Details

Use this task to view vBBU operational server information.

  1. From the navigation bar, select Devices > vBBU Details.
  2. In the vBBU Details window displays.
  3. In the vBBU Details window, click the Select vBBU drop-down list to select a vBBU.
  4. In the Software Details section, view the vBBU system fields that are described in the table below:

    Field

    Description
    Software Version Displays the vBBU software version.
    Software Build Date Displays when the vBBU software was released.
    System Boot Time Displays the last time the vBBU was rebooted.
    System Timezone Displays the timezone for where the vBBU system is located.
    Machine Type Displays the general type of server by processor. For example, x86_64.
    OS Name Displays the type of Linux operating system. For example, CentOs Linux.
    OS Release Displays the Linux operating system release. For example, 7.4.1708.
    The following example shows software details for a vBBU:
  5. In the Hardware Details section, view the vBBU system fields that are described in the table below:

    Field

    Description
    Server Displays the vBBU CPU and memory information.
    IPMI IP Displays the IP address of the Intelligent Platform Management Interface (IPMI) port on the vBBU server (management port of the server).
    Memory Displays how much memory the vBBU has in GB.
    The following example shows hardware details for a vBBU:
  6. In the vBBU Manager section, view the vBBU connection status to the PW EMS and its IP connectivity to northbound entities.

    Field

    Description
    Status Displays the connection status of the vBBU.
    vBBU IP Displays the IP address of the vBBU server.
    SON IP Displays the IP address of the Self-Organizing Network (SON) that manages the setup, optimization, performance, and availability of the wireless network.
    WAN IP Displays the Wide Area Network (WAN) Web IP address of the vBBU.
    The following example shows vBBU Manager details for a vBBU:
  7. In the vBBU Details window Operational Data section, you can click REFRESH to refresh the operational data columns that are described in the following table:

    Column

    Description
    Device Lists the vBBU and its associated vNodes.
    Status Lists the connectivity status of the vBBU and its associated vNodes.
    IP Address Lists the IP address of each vBBU and its associated vNodes.

  8. (Optional) Click Edit in the vBBU Manager section to view detailed vBBU information, perform vBBU system management actions, and modify vBBU and HNG system level parameters. These actions are discussed in further detail in the View vBBU Configuration Details section.
