Export a Dashboard

Use this task to export a dashboard from PW EMS Monitoring and Reporting.

  1. In the navigation bar, select the Dashboard icon .
  2. From the Dashboard pop-up menu, click Manage.
  3. In the Dashboards window that displays, select any existing dashboard that you want to export. For example: GSM_KPI.
  4. When the dashboard displays, select the share icon. For example:
  5. In the Share window, select the Export tab and click Save to file. For example:
    The dashboard json file is saved automatically to your Web browser default download location.
