Reboot a vNode

Use this task to reboot a vNode only or a vNode and its associated cells.

  1. Use any of the following ways to access the Device Summary window:
    • Log into the PW EMS, this window displays by default.
    • From the navigation bar, click Dashboard.
    • From the navigation bar, click Devices > Device Summary.
  2. Select the row of the vBBU device that you want to reboot, check the check box in this row, and click the Actions column button .
  3. In the pop-up menu, click VNode > Reboot as shown below
  4. In the confirmation dialog that appears, select VNode to reboot the vNode only or VNode+Cell to reboot both the vNode and its associated cells from the Reboot Type drop-down list. Next, click CONFIRM to perform the reboot action. For example:
