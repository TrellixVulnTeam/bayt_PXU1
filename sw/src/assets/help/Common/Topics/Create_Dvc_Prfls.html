Create Device Profiles

Use this chapter to create profiles for 2G and 4G devices.

Profiles can be used to manage configurations. The PW EMS provides several preexisting profiles that can be used for service provider 2G and 4G devices. These profiles contain default parameters that can be modified directly through the PW EMS, or downloaded from the PW EMS as a json file, which can then be modified and uploaded to the PW EMS. A default profile can be saved as a new profile and uploaded. This new profile can have different parameters based on the service provider network to which it is applied and be provided with a new name for its purpose or for what it is about. There is no limit to the number of profiles that can be created. When a profile is changed, it is propagated to all associated cells. Any changes to a profile that are made can be applied to a vBBU or cells immediately or scheduled to be applied to them at a later time to avoid disrupting service.

  • About GSM Profiles
    2G is the second generation cellular network technology for Global System for Mobile (GSM) communication.
  • About LTE Profiles
    4G is the fourth generation of cellular network technology that succeeds 3G capabilities and provides connectivity to most current Long-Term Evolution (LTE) cellphones.
  • View and Download Default GSM and LTE Profiles
    Use this task to view and download Global System for Mobile (GSM) profiles (2G) and Long-Term Evolution (LTE) profiles (4G) for service provider networks. Any profile that you download can be modified and uploaded later. Thereafter, this profile can be assigned and applied to vNodes.
  • Upload GSM and LTE Profiles
    Use this task to upload Global System for Mobile (GSM) profiles (2G) and Long-Term Evolution (LTE) profiles (4G) for service provider networks that you have previously downloaded and modified. Once, this profile is uploaded it can be given a new name and be assigned and applied to applicable vNodes.
  • Create a New Profile on the PW EMS
    Use this task to start creating a new GSM (2G) or LTE (4G) profile on the PW EMS from the beginning (greenfield).
  • Download an LTE or GSM Data Model Service Definition File
    Use this task to download the LTE (4G) or GSM (2G) data model service definition file to view descriptions in the json file to better understand how to configure 4G and 2G data model parameters and their values, modify default parameters in the LTE or GSM data model service definition files, or create new LTE or GSM data model service definition files from the defaults. A data model describes what parameters are service affecting or can be updated dynamically, such as the units, and so on.
