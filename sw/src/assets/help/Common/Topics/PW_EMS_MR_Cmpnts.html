PW EMS Monitoring and Reporting Components

The following table describes the PW EMS Monitoring and Reporting high-level functions that are used for the following purposes:

  • Collection - Pulls statistics data from the HetNet Gateway (HNG) using SFTP.
  • Network Health - Displays KPI data in flexible dashboards.
  • Device Health - Displays KPI data about network devices in flexible dashboards.
Function Description
Flexible dashboards
  • Support for various services within the 4G, mesh, and backhaul technologies
  • Customization to group by KPI classes or other characteristics
  • New dashboard creation using custom charts and views of existing data
  • Pre-defined dashboards included for important KPIs
  • Pre-defined groups provided you can copy and customize
Custom KPIs
  • Create new KPIs or modify pre-efined KPIs based on available counters
  • Immediately change existing KPIs and reports
Drill down capability
  • Drill-down by time or device
  • Define the order of dimensions according to your requirements
Data aggregation
  • Every chart has the option to statistically aggregate values
  • Aggegation is possible across any selected dimension, such as device / time (year / month / day / week / hour / timestamp)
Maintenance
  • Created dashboards can be exported locally
  • Exported files can be stored in a repository to maintain versioning
