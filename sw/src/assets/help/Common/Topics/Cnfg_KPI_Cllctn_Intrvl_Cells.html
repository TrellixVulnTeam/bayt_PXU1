Configure the KPI Collection Interval for Cells

Use this task to configure how often Key Performance Indicator (KPI) parameters are collected for 2G and 4G cells.

  1. From the navigation bar, click Settings > System.
  2. In the System Settings window, click Cell KPI and complete the following fields:

    Field

    Description
    2G Interval field Enter the 2G cell KPI collection interval from 5 to 1440 seconds. The default collection interval is 15 seconds.
    4G Interval field Enter the 4G cell KPI collection interval from 5 to 1440 seconds. The default collection interval is 15 seconds.
Changes to the KPI collection interval appear in the PW EMS Network Monitoring application. Refer to About PW EMS Network Monitoring section for more information.
