About GSM Profiles

2G is the second generation cellular network technology for Global System for Mobile (GSM) communication.

A 2G radio profile contains a grouping of parameters that form a basis for configuring or modifying a 2G device. The configuration or modification of a 2G device can only be accomplished by configuring or modifying a 2G profile. When any changes are made and saved, these changes are automatically applied to all the associated devices in the PW EMS database.

The PW EMS has one default 2G profile template called GSM Default that can be used. This profile template contains pre-defined parameters can be assigned and applied to one or more connected vBBUs on the service provider network. Also, a GSM profile can be built from the beginning (greenfield). Refer to the following sections for more information about using default and created GSM profiles in your environment.
