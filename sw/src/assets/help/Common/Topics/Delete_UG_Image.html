Delete a Device Software Image

Use this task to delete one or more device software images from the Device Upgrade and Rollback window.

  1. In the left navigation pane, click Devices > Upgrade.
  2. In the Device Upgrade and Rollback window, select the trash icon to the right of the software image file row in the Image Inventory table that you want to delete.
  3. In the Confirm Delete dialog box, click CONFIRM.
    The device software image is deleted from the Image Inventory table.
